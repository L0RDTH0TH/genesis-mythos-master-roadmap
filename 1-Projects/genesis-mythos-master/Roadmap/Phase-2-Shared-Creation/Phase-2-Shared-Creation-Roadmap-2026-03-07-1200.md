# Phase 2 — Shared creation

The generator is a collaborative forge: player lore intents (backstories, artifacts, faction ties) feed systemic hooks that become living elements in the world. Lore and narrative stay emergent; the systems that enable emergence are central (input pipelines, reputation/consequence graphs, dynamic triggers, feedback loops). Constraint: no hardcoded story — narrative seeds become mechanical reality through clear interfaces.

- [ ] Intent population pipeline: player/DM lore → reputation, events, environment state
- [ ] Modular input population and intent resolver for systemic hooks
- [ ] Reputation & consequence graphs; dynamic event triggers
- [ ] Feedback loops where player choices reshape simulation state over sessions
