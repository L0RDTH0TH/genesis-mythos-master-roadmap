# Phase 6 — Prototype scope (Q3 2026)

Target deliverable horizon: single-player playable world with one ruleset integrated, full perspective split (player first-person, DM free-cam + orthographic), live DM overwrites, one round-trip player intent loop, basic living simulation (weather + simple NPC schedules), and clear modding seams demonstrated (e.g. swap a biome generator or add a new event type). Success = proof of architecture and remixability.

- [ ] Single-player playable world with one ruleset integrated
- [ ] Full perspective split (player first-person, DM free-cam + orthographic)
- [ ] Live DM overwrites; one round-trip player intent loop
- [ ] Basic living simulation (weather + simple NPC schedules)
- [ ] Demonstrate modding seams (e.g. swap biome generator or add event type)
