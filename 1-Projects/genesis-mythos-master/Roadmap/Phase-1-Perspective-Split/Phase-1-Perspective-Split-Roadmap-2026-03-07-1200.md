# Phase 1 — Perspective split

Players live exclusively in first-person — the campaign is experienced, not watched. The DM commands with Sparky-inspired free-cam (orbit, pan, zoom, tilt) and can switch to orthographic/top-down for tactical work (tokens, fog, measurements). Mode switches use smooth interpolation so flow stays unbroken. Intent: pure immersion for players and commanding mastery for the DM.

- [ ] Lock player view to first-person only (no top-down/third-person escape)
- [ ] Implement DM free-flight camera (orbit, pan, zoom, tilt)
- [ ] Add orthographic/top-down toggle for tactical work
- [ ] Camera interpolator for mode switches (position, rotation, FOV easing)
