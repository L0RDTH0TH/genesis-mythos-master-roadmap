# Phase 3 — Living world

Vitality through layered simulation: weather that alters mood and paths, NPCs with intersecting agendas, ambient surprises, persistent scars from play. DM overwrites respect cost and intent — live tweaks for flair vs deliberate re-generation for structural changes. Extensibility allows swapping simulation flavors, visual styles, or rule behaviors without breaking cohesion. Goal: reward long campaigns with meaningful evolution.

- [ ] Tick-based simulation layer (weather, NPC schedules, factions, persistent state)
- [ ] DM overwrite rules: live tweaks vs structural re-generation
- [ ] Extensibility: swap simulation flavors, visual styles, rule behaviors
- [ ] Decouple simulation from rendering for mobile previews and scalability
