# Example companion amendment (pilot)

This note demonstrates the **Conceptual-Amendments** contract: **one file per section-level change** after the parent roadmap note is **frozen**. It is **not** a substantive design delta.

**Parent:** frontmatter `parent_roadmap_note` points at phase **4.1.1.10**. **`amends_section`** names the heading or `^block-id` in the parent that this amendment clarifies or overrides in natural language.
