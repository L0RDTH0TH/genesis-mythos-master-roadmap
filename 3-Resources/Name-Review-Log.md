# Name-Review-Log

Log for **NAME-REVIEW** queue mode (name-enhance batch). When EAT-QUEUE processes a NAME-REVIEW entry, append one line per note processed.

## Log line format

`YYYY-MM-DD HH:MM | note_path | suggested_name | applied: true/false | confidence: X% | protection_triggered: <value or -> | old_stem | #review-needed (if applicable)`

## Example

`2026-03-01 14:30 | 1-Projects/MyProject/Untitled 7.md | my-idea-about-x-2026-03-01.md | applied: true | confidence: 92% | protection_triggered: - | old_stem: Untitled 7`
