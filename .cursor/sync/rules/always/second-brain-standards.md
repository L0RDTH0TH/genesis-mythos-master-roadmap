# Second Brain Standards

- **PARA** method strictly: 1-Projects (time-bound), 2-Areas (ongoing responsibility), 3-Resources (evergreen reference), 4-Archives (inactive).
- All notes get searchable title, frontmatter, #tags.
- Atomic notes preferred — split if > 1–2 screens.
- Use `![[5-Attachments/PDFs/...]]` etc. for attachments, `[[ ]]` for note links.
- Note filenames: `kebab-slug-YYYY-MM-DD-HHMM.md` (slug first, date and time at end) per Naming-Conventions; daily/ingest date-time when created today.
