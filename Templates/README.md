# Templates

Organized template layout for the vault. See [[3-Resources/Second-Brain/Templates]] and [[3-Resources/Second-Brain/Template-Usage-Audit]] for full docs.

| Folder | Contents |
|--------|----------|
| **Decisions/** | Decision-Wrapper.md, Decision-Wrapper-Phase-Direction.md — used by ingest and EXPAND-ROAD pipelines. |
| **Roadmap/** | Master-Goal.md, Hand-Off-Roadmap.md, Planning-Prompt-Task.md, Roadmap-*-Template.md — roadmap and PMG. |
| **Ingest/** | Ingest-Selector.md, Ingest-Selector.tpl, Ingest-Template.md, By-Type/ (Stray-Thoughts, AI-Output, Link-Note). **Do not remove Ingest-Selector.tpl** — required for Templater. |
| **Notes/** | Session-Prep.md (Templater startup), Daily-Note-Template.md, Weekly-Note-Template.md, Mobile-AI-Question.md (Templater folder template for Ingest/Mobile-Questions). |
| **Chat/** | AI Prompts.md — copy-paste pipeline prompts; canonical phrases in 3-Resources/Second-Brain/Chat-Prompts.md. |
| **Prompt-Components/** | Base-Prompt, Param-Defaults, Param-Overrides, Guidance-Default, Error-Handling-Template, Skill-Chain — laptop prompt-crafter assembly. |
| **Scripts/** | ProcessIngestTrigger.js — user script (Templater `user_scripts_folder`); invoke via macro if needed. |
| **MDKit/** | Professional Markdown templates from [mdkit.io](https://mdkit.io): README, API docs, Changelog, Post-Mortem, Status Report, Meeting Minutes, SWOT, PRD, User Persona, Cornell Notes, Research Abstract, Literature Review, Blog Draft, Video Script, FAQ, Daily Planner, Habit Tracker, Gift List, Meal Plan, Travel Itinerary, Recipe, Resume. See [[Templates/MDKit/README]]. |
