# Agent Output — drop zone for Cursor agent output

Drop Markdown files produced by **Composer** or **Cursor cloud agents** here. These are typically dev-ready sub-tasks, phase breakdowns, or pseudo-code notes generated from master goal / roadmap phase notes.

**After dropping files here, run:** **INGEST MODE: batch on Ingest/Agent-Output/**

(Uses the documented batch ingest trigger from Pipelines.md / Queue-Alias-Table — no new alias needed.)

Notes in this folder (or with frontmatter `agent-generated: true`, `confidence_override: high`) may skip the Decision Wrapper and be moved directly to PARA when conditions in para-zettel-autopilot are met (see [[3-Resources/Second-Brain/Cursor-Agent-Ingest-Workflow]]).
