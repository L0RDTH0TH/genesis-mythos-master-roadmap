## Source: https://arxiv.org/html/2503.21474v2

The Procedural Content Generation Benchmark (PCG Benchmark) evaluates generative algorithms on 12 game-related problems with per-problem content representation and metrics for quality, diversity, and controllability. Framework follows OpenAI-Gym-style spaces for content and control parameters; evaluation returns percentages and per-artifact scores in [0,1] for each criterion. Intended as standardized comparison for PCGML, PCGRL, search-based PCG, etc. Repository: github.com/amidos2006/pcg_benchmark (per paper).

## Source: https://blog.aqd.is/2021/07/liar-pbt

Stateful Property-Based Testing extends stateless PBT by generating sequences of commands and checking properties at each step using an abstract state machine (simplified model) run in parallel with the real system; results are compared. Discusses LIAR prototype for EVE-style attribute modifiers across Erlang actors / locations. Explains duplication limits of TDD for interaction sequences and how PropCheck/PropEr command generation finds edge cases.
