# Archetype Menu System - Multi-Level Navigation

## Overview

The archetype submenu system has been expanded to support **4 levels of nested navigation**, allowing players to explore D&D character archetypes through a Jungian psychological framework with comprehensive subclass coverage.

## 📊 System Statistics (Updated 2025-10-30)

- **4** Main Categories
- **17** Jungian Archetypes
- **58** Class Options
- **245** Subclass Selections
- **4** Levels of Navigation Depth

## Navigation Hierarchy

### Level Structure

1. **Main Category** (4 options)
   - Identity and Self
   - Transformation and Balance
   - Guidance and Authority
   - Rebellion and Change

2. **Archetype** (17 total, 3-6 per category)
   - e.g., "The Self", "The Persona", "The Shadow"

3. **Class** (58 total, varies per archetype)
   - e.g., Monk, Paladin, Druid, Cleric

4. **Subclass** (245 total, 1-6 per class)
   - e.g., "Way of the Astral Self", "Way of Mercy"

## Implementation Details

### Files Modified
- `resources/test_menu_data.json` - Main menu with full archetype hierarchy
- `resources/archetype_menu_data.json` - Standalone archetype reference file

### Data Structure
The system uses the `MenuNode` class with recursive children:
- Each level can have any number of children
- Leaf nodes (subclasses) have empty children arrays
- Colors are defined per node using hex codes
- The fractal wheel automatically handles any depth of nesting

### Navigation Flow
1. **Main Menu** → Player → New Character
2. **Archetype Categories** (4 options radially displayed)
3. **Specific Archetypes** (varies per category)
4. **Classes** (thematically appropriate for chosen archetype)
5. **Subclasses** (final selection - triggers character creation)

## Color Coding Strategy

Colors are chosen thematically to reflect the psychological meaning:
- **Gold/Orange** - Unity, balance (The Self)
- **Blue** - Social adaptation (The Persona)
- **Black/Dark Gray** - Hidden aspects (The Shadow)
- **Purple/Pink** - Transformation, romance
- **Silver/Gray** - Wisdom, authority
- **Green** - Nature, nurturing
- **Brown** - Structure, protection
- **Red/Crimson** - Action, rebellion
- **Yellow** - Innocence, potential
- **Turquoise** - Creativity
- **Violet** - Magic, transformation

## Usage in Game

### For Players
1. Navigate through categories with mouse/keyboard
2. Click on archetype to see available classes
3. Click on class to see subclasses
4. Select subclass to finalize character concept
5. Press ESC at any level to go back

### For DMs
The archetype system helps guide players toward thematically coherent builds while exploring psychological depth in character creation.
