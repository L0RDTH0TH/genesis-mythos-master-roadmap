Build a 3d tree
Flat surface for growth 
Starting trunk made up of the following branches; dm, player, mods, and settings
Roots(base of trunk) will be exit.
Each branch of the tree will have children 
Each child lends weight to the branch
Trees is procedurally generated based of menu structure, picking realistic growth directions and lengths based on branch weight and user interaction 
Branches remain inactive until user selects them causing the selected branch to grow revealing more branches user progresses through menu options until weight is zero (no children) upon wich rest a leaf(s)
Leaves are input fields for user choices whether join game or backstory text field 
Camera should follow the growing branch along a track ensuring each new branch is in view with the ability to move backward down by clicking the main branch
