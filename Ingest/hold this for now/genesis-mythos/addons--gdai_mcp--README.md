# GDAI MCP Plugin

This directory is for the GDAI MCP Plugin addon.

## Installation

1. Download the plugin from: https://gdaimcp.com
2. Extract the plugin files into this directory (`addons/gdai_mcp/`)
3. Enable the plugin in Godot: Project > Project Settings > Plugins > Enable "GDAI MCP"
4. The plugin will provide an MCP config JSON snippet in the GDAI MCP panel - use that to configure your MCP client

## Documentation

Full installation and setup guide: https://gdaimcp.com/docs/installation
